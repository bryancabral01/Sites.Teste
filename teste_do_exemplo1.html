Programação, no contexto da tecnologia, é o processo de escrever um conjunto de instruções para um computador executar uma tarefa específica. Essas instruções são criadas em uma linguagem de programação, que serve como uma ponte de comunicação entre o ser humano e a máquina.

A programação se baseia no pensamento lógico e na criação de algoritmos, que são sequências de passos lógicos para resolver um problema. Os pilares do pensamento computacional que sustentam esse processo incluem:

Decomposição: Quebrar um problema complexo em partes menores e mais fáceis de resolver.

Reconhecimento de padrões: Identificar semelhanças entre os problemas.

Abstração: Focar nos detalhes mais importantes para a solução, ignorando os secundários.

Algoritmo: Criar a sequência de instruções exata para alcançar o objetivo.

Linguagens de programação são conjuntos formais de regras e instruções, como um "idioma", que permitem aos programadores criar programas de computador, aplicativos e outros softwares.

Elas funcionam como uma ponte entre a linguagem que os humanos entendem e a linguagem binária (código de máquina) que os computadores executam.

As linguagens variam de nível de abstração, como as de baixo nível (mais próximas do hardware), e as de alto nível (mais fáceis para os humanos lerem e escreverem), com exemplos populares que incluem HTML, CSS, Javascript, PHP e Phyton, cada uma com suas aplicações e vantagens.

HTML


HTML (HyperText Markup Language) é a linguagem de marcação usada para criar e estruturar páginas web. Ele define a estrutura do conteúdo, como títulos, parágrafos, imagens, links e tabelas.

CSS


CSS (Cascading Style Sheets) é a linguagem usada para estilizar e formatar páginas web. Ele controla cores, fontes, tamanhos, espaçamentos, layouts e muito mais, deixando o HTML visualmente bonito e organizado.

JAVASCRIPT


JavaScript (JS) é a linguagem de programação usada para tornar páginas web interativas. Com JS, você pode criar animações, validar formulários, manipular elementos da página, responder a cliques e eventos, entre muitas outras funcionalidades.

PHP


PHP (Hypertext Preprocessor) é uma linguagem de programação server-side usada para criar páginas web dinâmicas. Ele processa dados no servidor antes de enviar o conteúdo ao navegador, permitindo interações com banco de dados, formulários e sistemas de login.

PYTHON


Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Python é uma linguagem de programação versátil e de alto nível, usada para desenvolvimento web, automação, análise de dados, inteligência artificial, ciência de dados e muito mais. É conhecida por sua sintaxe simples e legível.

FIGURAS IMPORTANTES DA PROGRAMAÇÃO

LINUS TORVALDS

O próprio nome de Linus Torvalds já indica o impacto que ele teve no mundo da programação. Em 1991, da Universidade de Helsinki, ele criou o núcleo Linux, um dos principais expoentes do software livre e de código aberto. Curiosamente, Torvalds havia batizado sua criação como "Freax" (uma combinação de palavras entre "free", "freak" e "x"), mas o administrador do servidor optou por "Linux", que era o apelido que Linus usava na universidade. Hoje, milhares de programadores ao redor do mundo continuam contribuindo para o desenvolvimento do Linux. No entanto, a valiosa e fundamental contribuição inicial rendeu ao engenheiro de informática finlandês diversos prêmios e honrarias, como a inclusão no Hall da Fama da Internet, a nomeação pela revista Time como um dos heróis revolucionários dos últimos 60 anos, e até mesmo um asteroide com seu nome.

TIM BERNERS-LEE

Foi em 2004 que Tim Berners-Lee foi condecorado pela própria Rainha Isabel II com o título de Cavaleiro da Ordem do Império Britânico. Motivos para essa honra não faltam, é claro. Afinal, Tim foi o inventor do protocolo de transferência de hipertexto (sim, o HTTP), que é essencialmente a tecnologia na qual toda a World Wide Web se baseia. Sim, foi um divisor de águas, pois sua criação permitiu aos usuários acessar informações pela Internet. O impacto desse londrino perdura até hoje, pois ele ocupa a Cátedra de Fundadores no Laboratório de Informática e Inteligência Artificial do MIT (Massachusetts Institute of Technology) e é diretor da Alliance for Affordable Internet, uma organização cujo objetivo é tornar o acesso à Internet de banda larga muito mais acessível para todas as pessoas no mundo.

BJARNE STROUSTRUP

Nascido em 1950 na Dinamarca, Bjarne Stroustrup é atualmente professor na Cátedra de Informática da Faculdade de Engenharia da Universidade A&M do Texas, bem como na Universidade de Columbia. Seu currículo é altamente respeitável, uma vez que foi ele quem, em 1979, criou e desenvolveu a linguagem de programação C++, que é amplamente utilizada em todo o mundo e acrescentou novos recursos à linguagem C (criada por Dennis Ritchie). De fato, ele é autor de diversos livros, como "The C++ Programming Language", que é muito útil como texto introdutório para essa linguagem, "The Design and Evolution of C++", no qual aprofunda os princípios e limitações que moldaram o C++, e "Programming -- Principles and Practice Using C++", que oferece uma abordagem de programação que estabelece uma base sólida para escrever código de forma correta e eficiente.
